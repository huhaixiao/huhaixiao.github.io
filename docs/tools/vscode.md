# Shortcuts for macOS

- [https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf)

## Navigation

- Show all Symbols
    
    ⌘ t
    
- Go to File
    
    ⌘ p
    
- Go back/forward
    
    ⌃ - / ⌃ ⇧ -
    
- Go to Line
    
    ⌃ g
    
- Go to Symbol
    
    ⇧ ⌘ o
    
- Go to next/previous error or warning
    
    F8 / ⇧ F8
