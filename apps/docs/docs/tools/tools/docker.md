- What is docker?

- Virtual Machines vs Containers
    - Virtual Machines Problems
        1. Each VM needs a full-blown OS
        2. Slow to start
        3. Resource intensive
    - Container Pros
        1. Allow running multiple apps in isolation
        2. Are lightweight
        3. Uses OS of the host
        4. Start quickly
        5. Need less hardware resources
